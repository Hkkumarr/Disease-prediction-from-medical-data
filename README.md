# 🩺 Disease Prediction System Using Machine Learning

## 📌 Project Overview

The Disease Prediction System is a Machine Learning web application developed using Streamlit and Random Forest Classifier. The system predicts whether a patient is likely to have diabetes based on medical attributes such as glucose level, BMI, age, insulin level, and other health indicators.

## 🚀 Features

- Interactive Streamlit Web Application
- Diabetes Prediction using Machine Learning
- Random Forest Classification Model
- Dataset Visualization
- Confusion Matrix Visualization
- Feature Importance Analysis
- Real-Time Patient Prediction
- Model Performance Evaluation

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## 📊 Dataset

The project uses the Diabetes Dataset containing the following features:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Target Variable)

## 🤖 Machine Learning Model

Algorithm Used:

- Random Forest Classifier

Why Random Forest?

- High Accuracy
- Handles Complex Data Well
- Reduces Overfitting
- Provides Feature Importanc

## 📈 Application Workflow

1. Load Diabetes Dataset
2. Split Dataset into Training and Testing Sets
3. Train Random Forest Model
4. Evaluate Model Performance
5. Display Accuracy and Confusion Matrix
6. Accept Patient Health Information
7. Predict Diabetes Risk
8. Display Prediction Probability
